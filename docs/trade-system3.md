## 1. 4h周期：定义战场与规则（新增震荡与转折判定）

**核心逻辑：** 4h 不仅决定方向，更决定**“期望值”**和**“持仓时间”**。

1. **判断市场状态（Context Identification）：**

   - **A. 强趋势（Trend）：** 高点抬高（HH）+ 低点抬高（HL），且动能（RSI/MACD）未衰竭。
   - **B. 震荡（Range）：** 价格在一个大区间内上下扫荡。
   - **B+. 震荡区间突破（Range）：** 价格突破震荡区间
   - **C. 混乱（Chop）：** 非趋势区间，结构转换阶段，比如跌破了上涨趋势的strong low形成了choch，随后价格随意移动。危险形态不交易。只观察

2. **根据状态制定基调：**
   - **趋势基调：** 只做顺势。寻找回调入场。**（胜率最高，盈亏比最大）**
   - **震荡基调：**
     - 将震荡区间划分为 **溢价区（上25%）**、**均衡区（中间50%）**、**折价区（下25%）**。
     - **规则：** 严禁在均衡区交易。只在溢价区找空，只在折价区找多。
     - **心态：** 即使 4h 看起来像要突破，在未实际收盘突破前，默认突破会失败（Fakeout）。
   - **混乱基调：**
     - 找到更大时间范围的阻力与支撑位置，等待价格到这些地方测试，确认新的趋势诞生。
     - 等待波动逐渐稳定在某个区间内。形成震动区间

## 2. 绘制波浪与关键结构

1. 标记 Swing High / Swing Low。
2. **震荡特供：** 标记 Range High (RH) 和 Range Low (RL)，以及中轴 (EQ)。

## 3. 1h周期+15min周期观察具体信号：根据4h背景选择作战模式（顺势/区间）

在此步骤，必须根据第1步的背景选择不同的 1h 结构：

**时间分工（统一口径）：**

- **1h：主要看结构/位置。** 用来判断模式与关键位（趋势/震荡、是否到达 RH/RL/关键位、是否发生 BOS/CHOCH）。
- **15min：主要看价格行为。** 所有触发类信号（SFP、Pinbar/锤子线、吞没、节奏变慢、停止量/VSA）都以 15min 为准。
- **Footprint：核心滤网。** 用来确认吸收/失衡/主动性回归，避免“看起来像”的假信号。

### 模式 A：顺势模式（4h 趋势强劲）

- **寻找结构：** 1h 出现与 4h 同向的结构。
- **形态：** Bull/Bear Flag, 楔形, 三角形。
- **信号：** 等待回调结束的信号（BOS, 趋势线突破）。
- **记录方式（与 B / B+ 一致）：** 用“计分卡表格”强制记录入场理由，避免只靠感觉下单。

| 权重           | 信号类型                  | 具体指标/现象                                                               | 你的检查动作                                                             |
| :------------- | :------------------------ | :-------------------------------------------------------------------------- | :----------------------------------------------------------------------- |
| **必须**       | **背景 (Context)**        | 4h 趋势清晰（HH+HL 或 LH+LL），未出现趋势失效信号                           | **看：4h。** 先确认：这是“顺势续航”，不是“趋势末端”。                    |
| **必须**       | **位置 (Location)**       | 回撤进入关键支撑/阻力带（如 0.236–0.618、关键结构位、订单密集区）           | **看：4h + 1h。** 没到关键位不做；到了才开始找入场。                     |
| **1分**        | **结构 (Structure)**      | 15min/1h 出现顺势的 BOS / CHOCH / 趋势线突破                                | **看：1h（结构）+ 15min（触发）。** 结构确认要清晰，避免在回撤途中硬接。 |
| **1分**        | **节奏 (Tempo)**          | **速度变慢**：回撤过程中 K 线实体变小/波幅收敛，或出现长下影/长上影（拒绝） | **看：1h。** 问自己：“下跌/上涨还顺畅吗？”不顺畅才考虑切入。             |
| **1分**        | **K线 (Candle)**          | 关键位出现 Pinbar / 吞没 / 破底翻（或破顶翻）                               | **看：15min。** 只把 K 线当“触发器”，必须发生在正确位置。                |
| **1分**        | **背离 (Divergence)**     | Hidden Divergence（隐藏背离）或动能回升迹象                                 | **看：1h（指标/动能）。** 作为加分项，不单独构成入场理由。               |
| **2分 (核心)** | **订单流 (Continuation)** | 顺势方向出现 Imbalance/主动性回归，或回撤端出现吸收后转强                   | **看：Footprint 3 5 15min。** 看到主动性回归/转强，才算真正续航。        |

- **开仓规则（建议）：** 总分 **≥ 3分** 且包含 **至少一个核心项** 才允许开单。
- **止损：** 结构失效点外侧（如回调 swing low/high 外侧或触发 K 线针尖外侧）。
- **止盈：** 首目标前高/前低或关键结构位；趋势延续则分批止盈/移动止损。

### 模式 B：震荡模式（4h 处于震荡）

- **位置限制：** 价格必须处于 4h 的溢价区（卖出）或折价区（买入）。
- **寻找结构：**
  - **Liquidity Sweep (流动性猎取)：** 1h 假突破 4h 区间边缘后迅速收回。
  - **CHOCH (结构改变)：** 在边缘出现小级别的结构破坏。
- **核心信号：** **努力 vs 结果（Effort vs Result）**。当出现**超高成交量/Delta（努力很大）**，但价格却**不再推进/无法有效跌破（结果很弱）**，往往意味着边界处出现了**吸收（Absorption）/停止量（Stopping Volume）**。

| 权重           | 信号类型                   | 具体指标/现象                                                      | 你的检查动作                                                              |
| :------------- | :------------------------- | :----------------------------------------------------------------- | :------------------------------------------------------------------------ |
| **必须**       | **位置 (Location)**        | 价格触及 **RH / RL**（溢价区做空，折价区做多）                     | **看：4h + 1h。** 不到边界绝不动手；EQ（中轴）区域禁止交易。              |
| **1分**        | **形态 (Pattern)**         | **SFP / Liquidity Sweep（假突破回收）**                            | **看：15min。** 刺破边界后，收盘价快速回到区间内；越“干净”越好。          |
| **1分**        | **K线 (Candle)**           | **Pinbar / 锤子线 / 吞没**（长影线拒绝）                           | **看：15min。** 实体可以是红/绿，重点是影线极长且收回区间。               |
| **1分**        | **回测确认 (Retest Hold)** | 回到区间内部后，回踩 RL（或下沿）未有效跌破，并出现拒绝/小结构转强 | **看：15min。** “回踩不破 + 重新站回”，优先级高于单根 K 线好看。          |
| **2分 (核心)** | **量价 (VSA)**             | **Stopping Volume（停止量）**：超高量 + 价格不再推进               | **看：15min + 成交量/Delta。** 努力最大但结果很弱（站不出去/站回来了）。  |
| **2分 (核心)** | **订单流 (Absorption)**    | 激进买/卖单被被动墙吸收，价格却走不动                              | **看：Footprint 3 5 15min。** Delta 很大但价格不继续推进。                |
| **1分**        | **背离 (Divergence)**      | RSI/MACD 背离 或 Delta 背离                                        | **看：15min（指标）+ Footprint（Delta）。** 作为增强项：动能/主动性衰竭。 |

- **开仓规则（建议）：** 总分 **≥ 3分** 且包含 **至少一个核心项**。
- **入场示例（折价区做多）：** 触及 RL (必须) + Pinbar/锤子线 (1分) + Stopping Volume (2分) + SFP (1分) = **4分（高胜率配置）**。

**风险提示（“红 K 线的诅咒”）：**

- 如果你是在“长下影但实体仍为阴线/红 K”收盘直接进场，这是**激进侧入场**，存在动能惯性导致下一根继续下杀的风险。

**更稳健的入场（下一根确认法）：**

- 等待下一根 15min K 线确认。
- 做多：下一根转阳，且突破锤子线的收盘价或高点再入场；做空反之。

* **止损：** 放在针尖（影线最低/最高）外侧一点点；若“停止量”都被击穿，必须果断止损。
* **止盈：** 第一目标 EQ/POC（区间中轴），第二目标对侧边界（RH/RL）；到达后可推保护至保本。

### 模式 B+：区间突破（4h 震荡区间）

**目标：** 确认震荡结束，跟随新方向。
**心态：** “这次狼真的来了，别挡路。”
**注意：** 严禁追第一根突破 K 线，只做 **Break & Retest (突破回踩)**。

| 权重           | 信号类型                  | 具体指标/现象           | 你的检查动作                                                       |
| :------------- | :------------------------ | :---------------------- | :----------------------------------------------------------------- |
| **必须**       | **强力引发 (Initiation)** | **1h/4h 实体饱和 K 线** | **看：4h/1h 收盘。** 收盘价必须清晰地站稳在区间之外（无长影线）。  |
| **1分**        | **价值接受 (Acceptance)** | 突破后在区间外盘整      | **看：1h。** 价格没有立即跌回区间，而在外部形成新的成交密集区。    |
| **1分**        | **阻力互换 (Flip)**       | 缩量回踩区间边缘        | **看：15min（价格行为）+ 成交量。** 慢回踩、缩量、形态干净。       |
| **2分 (核心)** | **订单流 (Imbalance)**    | **激进引发 / 失衡链**   | **看：Footprint。** 突破 K 线内部有连续的 Imbalance (买方失衡)。   |
| **2分 (核心)** | **订单流 (Support)**      | **回踩出现支撑**        | **看：Footprint + 15min。** 回踩出现支撑/Delta 翻转/被动买单托底。 |

- **入场示例：** 4h大阳线收在区间外 (必须) + 回踩原阻力位 (1分) + 回踩时缩量且 Footprint 出现 Delta 翻转 (2分) = **3分 (开多)**。
- **止损：** 跌回震荡区间内部（说明突破失败，转为假突破逻辑）。
- **止盈：** 斐波那契扩展 1.618 或下一个大级别结构位。

## 4. 订单流（Footprint）：最终滤网与确认（不同场景权重不同）

### 场景 A：顺势交易（动能强）

- **角色：** 辅助验证。
- **关注点：** 是否有持续的失衡（Imbalance）堆叠推动价格？
- **宽容度：** 只要没有明显的反向阻力（吸收），即可入场。

### 场景 B：震荡（风险高，必须严苛）

- **角色：** **强制滤网（Mandatory Filter）。** 如果没有订单流确认，禁止入场。
- **必须看到的信号（任一）：**
  1. **吸收（Absorption）：** 价格冲击 4h 区间边缘，Buy Delta 很大但价格不涨（被动限价卖单吸收）。
  2. **Delta 背离（Delta Divergence）：** 价格创新高，但 Delta 显著下降（买盘枯竭）。
  3. **失衡反转（Failed Imbalance）：** 之前出现的买方失衡被瞬间吞没并形成卖方失衡。
- **案例应用（如你图所示）：**
  - 在震荡区间顶部（4h 阻力），15min K线收出上影线。
  - 查看 3min 足迹图：发现顶部有大量买单成交（绿色），但价格收在下方，且 Delta 出现背离。
  - **结论：** 这是一个完美的“震荡猎杀”做空信号。
