## 1. 4h周期：定义战场与规则（新增震荡与转折判定）

**核心逻辑：** 4h 不仅决定方向，更决定**“期望值”**和**“持仓时间”**。

1. **判断市场状态（Context Identification）：**

   - **A. 强趋势（Trend）：** 高点抬高（HH）+ 低点抬高（HL），且动能（RSI/MACD）未衰竭。
   - **B. 震荡（Range）：** 价格在一个大区间内上下扫荡。
   - **B+. 震荡区间突破（Range）：** 价格突破震荡区间
   - **C. 混乱（Chop）：** 非趋势区间，结构转换阶段，比如跌破了上涨趋势的strong low形成了choch，随后价格随意移动。危险形态不交易。只观察

2. **根据状态制定基调：**
   - **趋势基调：** 只做顺势。寻找回调入场。**（胜率最高，盈亏比最大）**
   - **震荡基调：**
     - 将震荡区间划分为 **溢价区（上25%）**、**均衡区（中间50%）**、**折价区（下25%）**。
     - **规则：** 严禁在均衡区交易。只在溢价区找空，只在折价区找多。
     - **心态：** 即使 4h 看起来像要突破，在未实际收盘突破前，默认突破会失败（Fakeout）。
   - **混乱基调：**
     - 找到更大时间范围的阻力与支撑位置，等待价格到这些地方测试，确认新的趋势诞生。
     - 等待波动逐渐稳定在某个区间内。形成震动区间

## 2. 绘制波浪与关键结构

1. 标记 Swing High / Swing Low。
2. **震荡特供：** 标记 Range High (RH) 和 Range Low (RL)，以及中轴 (EQ)。

## 3. 1h周期：根据4h背景选择作战模式（顺势/区间）

在此步骤，必须根据第1步的背景选择不同的 1h 结构：

### 模式 A：顺势模式（4h 趋势强劲）

- **寻找结构：** 1h 出现与 4h 同向的结构。
- **形态：** Bull/Bear Flag, 楔形, 三角形。
- **信号：** 等待回调结束的信号（BOS, 趋势线突破）。
- **信号：** 等待回调结束的信号（BOS, 趋势线突破）。
  - 位置对吗？ 是否到了 4h 的支撑0.236-0.618 区域？（没到坚决不看）
  - 速度慢了吗？ 跌到这里时，K 线变小了吗？或者出现长下影线了吗？
  - 微观有信号吗？ 15min 出现看涨吞没/CHOCH 了吗？
  - 订单流有吸收吗？ 是否有巨量支撑？
  - 如果满足 3 个以上，直接设好止损开单。
- **背离：** 寻找 Hidden Divergence（隐藏背离）作为顺势加分项。

### 模式 B：震荡模式（4h 处于震荡）

- **位置限制：** 价格必须处于 4h 的溢价区（卖出）或折价区（买入）。
- **寻找结构：**
  - **Liquidity Sweep (流动性猎取)：** 1h 假突破 4h 区间边缘后迅速收回。
  - **CHOCH (结构改变)：** 在边缘出现小级别的结构破坏。
- **背离：** 常规 RSI/MACD 背离是震荡反转的最强信号。
- **信号：**
  | 权重 | 信号类型 | 具体指标/现象 | 你的检查动作 |
  | :--- | :--- | :--- | :--- |
  | **必须** | **位置 (Location)** | 价格触及 Range High 或 Range Low | 不到边界绝不动手。 |
  | **1分** | **形态 (Pattern)** | **SFP (Swing Failure Pattern)** | 刺破边界后，K线收盘价收回区间内。 |
  | **1分** | **K线 (Candle)** | 15min 收出 **Pinbar (长影线)** 或 吞没形态 | 只有影线没有实体，说明拒绝。 |
  | **1分** | **背离 (Divergence)** | RSI 背离 或 **Delta 背离** | 价格新高/低，但动能/买卖盘减弱。 |
  | **2分 (核心)** | **订单流 (Absorption)** | **被动墙 / 吸收** | Footprint 显示顶部有大量 Delta 堆积（如绿色），但价格不涨。 |
  | **2分 (核心)** | **订单流 (Zero Print)** | **买盘/卖盘枯竭** | 极速回落，顶部没有成交量（0 Print），说明是真空拉升。 |

* **入场示例：** 触及 RH (必须) + 收出长上影 SFP (1分) + Footprint 顶部有 500 个买单被吸收 (2分) = **3分 (开空)**。
* **止损：** 放在针尖外侧一点点。
* **止盈：** 第一目标 EQ，第二目标 对侧 RL。

### 模式 B+：区间突破（4h 震荡区间）

**目标：** 确认震荡结束，跟随新方向。
**心态：** “这次狼真的来了，别挡路。”
**注意：** 严禁追第一根突破 K 线，只做 **Break & Retest (突破回踩)**。

| 权重           | 信号类型                  | 具体指标/现象           | 你的检查动作                                                        |
| :------------- | :------------------------ | :---------------------- | :------------------------------------------------------------------ |
| **必须**       | **强力引发 (Initiation)** | **1h/4h 实体饱和 K 线** | 收盘价必须清晰地站稳在区间之外（无长影线）。                        |
| **1分**        | **价值接受 (Acceptance)** | 突破后在区间外盘整      | 价格没有立即跌回区间，而在外部形成新的成交密集区。                  |
| **1分**        | **阻力互换 (Flip)**       | 缩量回踩区间边缘        | 价格慢慢跌回原阻力位，成交量萎缩。                                  |
| **2分 (核心)** | **订单流 (Imbalance)**    | **激进引发 / 失衡链**   | 突破的那根 K 线内部有连续的 Imbalance (买方失衡)。                  |
| **2分 (核心)** | **订单流 (Support)**      | **回踩出现支撑**        | 回踩时没有大量反向卖单，且出现 Delta 翻转（红变绿）或被动买单支撑。 |

- **入场示例：** 4h大阳线收在区间外 (必须) + 回踩原阻力位 (1分) + 回踩时缩量且 Footprint 出现 Delta 翻转 (2分) = **3分 (开多)**。
- **止损：** 跌回震荡区间内部（说明突破失败，转为假突破逻辑）。
- **止盈：** 斐波那契扩展 1.618 或下一个大级别结构位。

## 4. 订单流（Footprint）：最终滤网与确认（不同场景权重不同）

### 场景 A：顺势交易（动能强）

- **角色：** 辅助验证。
- **关注点：** 是否有持续的失衡（Imbalance）堆叠推动价格？
- **宽容度：** 只要没有明显的反向阻力（吸收），即可入场。

### 场景 B：震荡（风险高，必须严苛）

- **角色：** **强制滤网（Mandatory Filter）。** 如果没有订单流确认，禁止入场。
- **必须看到的信号（任一）：**
  1. **吸收（Absorption）：** 价格冲击 4h 区间边缘，Buy Delta 很大但价格不涨（被动限价卖单吸收）。
  2. **Delta 背离（Delta Divergence）：** 价格创新高，但 Delta 显著下降（买盘枯竭）。
  3. **失衡反转（Failed Imbalance）：** 之前出现的买方失衡被瞬间吞没并形成卖方失衡。
- **案例应用（如你图所示）：**
  - 在震荡区间顶部（4h 阻力），15min K线收出上影线。
  - 查看 3min 足迹图：发现顶部有大量买单成交（绿色），但价格收在下方，且 Delta 出现背离。
  - **结论：** 这是一个完美的“震荡猎杀”做空信号。
