这是一个经过深度思考和逻辑重构的 **《全天候日内交易系统 v3.0》**。

这一版本基于“奥卡姆剃刀”原则，剔除了原本模糊的“转折模式（Mode C）”，将市场简化为二元状态：**趋势（不平衡）**与**震荡（平衡）**，并将交易重心放在**“边界行为”**的识别上（是反转回震荡，还是突破开启新趋势）。

# 全天候日内交易系统 v3.0（二元逻辑：趋势 + 震荡 + 相变）

**核心哲学：** 市场只有两种状态——**行进中（Trend）** 和 **休息中（Range）**。我们要么跟随行进，要么在休息区的边缘高抛低吸，要么捕捉休息结束后的爆发。

---

## 1. 4h 周期：定义战场与结构识别

在开单前，必须先在 4h 图表上回答：**“现在的市场是在跑，还是在跳？”**

### A. 趋势状态 (Trend Mode)
- **定义：** 清晰的高点抬高（HH+HL）或低点降低（LH+LL）。
- **策略：** **Mode A（顺势回调）**。
- **失效：** 当价格破坏关键结构点（CHOCH）时，趋势暂停，进入“混乱观察期”。

### B. 震荡状态 (Range Mode)
- **定义：** 价格在一个大区间内扫荡，没有明确方向，VPVR 显示明显的 D 形分布。
- **策略：** **Mode B（震荡猎杀）**。

### C. 混乱/重置状态 (Reset / Chaos)
- **定义：** 刚刚发生 CHOCH（趋势结构破坏），但尚未形成新的震荡区间。
- **策略：** **空仓观察**。等待新的高点和低点确立，画出新的“笼子（区间）”后，转为震荡模式交易。

---

## 2. 绘制关键位（绘制笼子）

1.  **Swing Points：** 标记 4h 的有效高低点。
2.  **震荡三线（必须画）：**
    -   **RH (Range High / 阻力)：** 卖方防守位（溢价区）。
    -   **RL (Range Low / 支撑)：** 买方防守位（折价区）。
    -   **EQ (Equilibrium / 中轴)：** 50% 位置或 POC（控制点）。**（绝对禁止开单区）**

---

## 3. 1h 周期：选择作战模式

根据价格相对于 4h 结构的位置，选择对应的剧本：

### 模式 A：顺势回调 (Trend Continuation)
- **场景：** 4h 趋势完好，价格回撤至支撑/阻力区。
- **逻辑：** 顺水推舟。

### 模式 B：震荡猎杀 (Range Fade)
- **场景：** 价格触碰 4h 震荡区间的 **RH** 或 **RL**。
- **逻辑：** 假设突破会失败，价格回归均值（EQ）。

### 模式 B+：区间突破 (Range Breakout)
- **场景：** 价格强力冲破 4h 震荡区间，并发生“性质改变”。
- **逻辑：** 震荡结束，新的单边趋势（Mode A）诞生。

## 4. 15min + 订单流：入场信号计分卡（核心更新）

这是本系统的核心执行模块。为了提高胜率，我们采用**“多权重计分制”**。
**规则：** 总分 **≥ 3分** 且 **必须包含“核心滤网”** 才能扣动扳机。

### 场景一：模式 B（震荡猎杀 / 假突破）

**目标：** 在区间边缘做反向（高抛低吸）。
**心态：** “这是个陷阱。”

| 权重 | 信号类型 | 具体指标/现象 | 你的检查动作 |
| :--- | :--- | :--- | :--- |
| **必须** | **位置 (Location)** | 价格触及 Range High 或 Range Low | 不到边界绝不动手。 |
| **1分** | **形态 (Pattern)** | **SFP (Swing Failure Pattern)** | 刺破边界后，K线收盘价收回区间内。 |
| **1分** | **K线 (Candle)** | 15min 收出 **Pinbar (长影线)** 或 吞没形态 | 只有影线没有实体，说明拒绝。 |
| **1分** | **背离 (Divergence)** | RSI 背离 或 **Delta 背离** | 价格新高/低，但动能/买卖盘减弱。 |
| **2分 (核心)** | **订单流 (Absorption)** | **被动墙 / 吸收** | Footprint 显示顶部有大量 Delta 堆积（如绿色），但价格不涨。 |
| **2分 (核心)** | **订单流 (Zero Print)** | **买盘/卖盘枯竭** | 极速回落，顶部没有成交量（0 Print），说明是真空拉升。 |

*   **入场示例：** 触及 RH (必须) + 收出长上影 SFP (1分) + Footprint 顶部有 500 个买单被吸收 (2分) = **3分 (开空)**。
*   **止损：** 放在针尖外侧一点点。
*   **止盈：** 第一目标 EQ，第二目标 对侧 RL。

---

### 场景二：模式 B+（区间突破 / 真突破）

**目标：** 确认震荡结束，跟随新方向。
**心态：** “这次狼真的来了，别挡路。”
**注意：** 严禁追第一根突破 K 线，只做 **Break & Retest (突破回踩)**。

| 权重 | 信号类型 | 具体指标/现象 | 你的检查动作 |
| :--- | :--- | :--- | :--- |
| **必须** | **强力引发 (Initiation)** | **1h/4h 实体饱和 K 线** | 收盘价必须清晰地站稳在区间之外（无长影线）。 |
| **1分** | **价值接受 (Acceptance)** | 突破后在区间外盘整 | 价格没有立即跌回区间，而在外部形成新的成交密集区。 |
| **1分** | **阻力互换 (Flip)** | 缩量回踩区间边缘 | 价格慢慢跌回原阻力位，成交量萎缩。 |
| **2分 (核心)** | **订单流 (Imbalance)** | **激进引发 / 失衡链** | 突破的那根 K 线内部有连续的 Imbalance (买方失衡)。 |
| **2分 (核心)** | **订单流 (Support)** | **回踩出现支撑** | 回踩时没有大量反向卖单，且出现 Delta 翻转（红变绿）或被动买单支撑。 |

*   **入场示例：** 4h大阳线收在区间外 (必须) + 回踩原阻力位 (1分) + 回踩时缩量且 Footprint 出现 Delta 翻转 (2分) = **3分 (开多)**。
*   **止损：** 跌回震荡区间内部（说明突破失败，转为假突破逻辑）。
*   **止盈：** 斐波那契扩展 1.618 或下一个大级别结构位。

## 5. 全流程执行检查表 (SOP)

在每一次点击鼠标前，按顺序通过此流程：

1.  **宏观定性 (4h)：**
    -   现在是趋势（Mode A）还是震荡（Mode B）？
    -   如果是 CHOCH 后的混乱期，**是否已经空仓等待**？

2.  **微观定位 (1h)：**
    -   如果 Mode B：价格是否到了 RH (卖) 或 RL (买)？（不到不做）
    -   如果 Mode B+：价格是否已经完成了“突破 + 站稳 + 回踩”？（不回踩不做）

3.  **信号确认 (15min + Footprint)：**
    -   **计分环节：** 根据上文表格，我的入场理由得分是否 **≥ 3分**？
    -   **致命一问：** 订单流是否支持我的假设？（震荡要有吸收，突破要有失衡）。

4.  **风控设定：**
    -   震荡单：盈亏比至少 1:2（止损很窄，目标 EQ）。
    -   突破单：盈亏比至少 1:3（止损较宽，目标新趋势）。

---

### 总结：你的进化点

*   **删繁就简：** 删除了易混淆的 Mode C，不再主观臆测“反转”。
*   **量化标准：** 为最难做的震荡和突破引入了 **“计分制”**，强制要求订单流（Footprint）介入，杜绝单纯看 K 线形态的假信号。
*   **逻辑闭环：** 震荡边界只有两个结果——要么被挡住（Mode B 策略），要么冲出去（Mode B+ 策略）。系统覆盖了所有可能性。
